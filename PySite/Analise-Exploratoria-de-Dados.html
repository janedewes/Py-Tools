Análise Exploratória de Dados com Python¶

Anlise exploratria em um dos datasets para Machine Learning, o dataset iris com informaes sobre 3 tipos de plantas.

Esse dataset comumente usado em problemas de Machine Learning de classificao, quando nosso objetivo prever a classe dos dados. No caso deste dataset, devemos prever a categoria de uma planta a partir de medidas da planta (sepal e petal).

Dataset (j disponvel com o Scikit-Learn): https://archive.ics.uci.edu/ml/datasets/iris
